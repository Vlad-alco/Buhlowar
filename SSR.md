## 🔧 Компоненты системы

### 1️⃣ Твердотельное реле (SSR) — модуль управления

| Параметр | Значение |
|----------|----------|
| **Тип** | SSR с опторазвязкой, Zero-Cross |
| **Управление** | 5 В DC, активация высоким уровнем (0–2.5 В = ON) |
| **Нагрузка** | 24–240 В AC, до 2 А (резистивная) |
| **Опторазвязка** | ≥2.5 кВ |
| **Подключение** | Винтовые клеммы, компактный корпус |
| **Пример** | Модуль на базе Omron G3MB-202P или аналог |

✅ **Преимущества для проекта:**
- Прямое подключение к GPIO ESP32 (3.3 В логика совместима)
- Бесшумная работа (нет механических щелчков)
- Zero-Cross переключение снижает помехи в сети
- Гальваническая развязка защищает плату от сетевых наводок

⚠️ **Ограничения:**
- Номинал 2 А — для **резистивной** нагрузки; для индуктивной (клапан) нужен запас
- При токе >0.5 А возможен нагрев — обеспечить вентиляцию
- Не предназначен для фазового управления (диммеры)

### 2️⃣ Модуль защиты — RC-снаббер + варистор

| Параметр | Значение |
|----------|----------|
| **Тип** | Готовый модуль: RC-цепочка + варистор |
| **Напряжение** | 5–400 В AC/DC |
| **Мощность** | до 1000 Вт |
| **Размеры** | 31 × 13 мм |
| **Подключение** | Винтовые клеммы, параллельно нагрузке или SSR |

🔹 **Назначение:**
- **RC-цепочка**: гасит выброс напряжения (back-EMF) при размыкании индуктивной нагрузки
- **Варистор**: шунтирует импульсные перенапряжения >470 В (сетевые помехи, грозовые выбросы)

🔹 **Как подключать:**
220В ──► [SSR] ──► [Клапан] ──► 220В
│ │
[RC-модуль] ← параллельно клапану (рекомендуется)
│ │


✅ Один модуль — на один клапан. Полярность для AC не важна.

### 3️⃣ Электромагнитный клапан — нагрузка

**Модель:** `HOENYZY VX2120-08-SS-NO` (или аналог)

| Параметр | Значение |
|----------|----------|
| **Тип** | Нормально закрытый (NC) ✅ |
| **Напряжение катушки** | AC 220 В |
| **Присоединение** | 1/4" (DN8), резьба |
| **Отверстие** | 2.5 мм |
| **Корпус** | Нержавеющая сталь |
| **Рабочее давление** | 0–10 бар |
| **Среда** | Вода, пар, воздух, масло, спиртовые пары* |
| **Уплотнение** | **Проверить!** Заявлено FKM (Viton), но может быть силикон |

⚠️ **Критически важно: материал уплотнения**

❌ Силикон:
• Разрушается от спиртовых паров
• Теряет эластичность при >80 °С
• Риск протечки через 10–50 часов работы
✅ Viton (FKM):
• Химически стойкий к спирту, кислотам
• Рабочая температура до 200 °С
• Ресурс 1000+ часов
🔎 Как проверить:
Цвет: Viton — чёрный/тёмно-коричневый, силикон — прозрачный/белый
Тест спиртом: капля 96% спирта — если разбухает за 10 мин, это не Viton
При сомнении — заменить уплотнение на витоновое (ремкомплекты доступны)


🔹### **Электрические параметры катушки (расчётные):**

 Сопротивление: 1.5–3.5 кОм (измерить мультиметром)
• Мощность: 8–15 ВА
• Рабочий ток: ~40–70 мА
• Пусковой ток: ~150–250 мА (в 3× больше рабочего)


✅ **Совместимость с SSR:** запас по току >10× — отлично!

## 🔌 Схема подключения

ESP32 GPIO ─[220Ω]─►│ SSR-модуль (5В, low trigger) │
GND ───────────────►│ Вход: 0–2.5В = ON │
│ │
│ Выход AC: │
220В L ─[Предохр.1А]─►│ AC1 ──►[Клапан]──► AC2 │
│ │ │ │
│ [RC-модуль] │
│ │ │ │
220В N ─────────────►│ └────┘ │
│ │
└─────────────────────────────────┘
🔧 Дополнительно:
• Заземление металлического корпуса клапана (если есть)
• Термостойкие провода ПВС 0.5–0.75 мм²
• Герметичный ввод проводов в корпус электроники
• Предохранитель 1 А на фазе — защита от КЗ

📐 Расчёт и совместимость
Проверка токовой нагрузки

Клапан: 220 В, ~10 ВА → I_раб = 10 / 220 ≈ 0.045 А
Пусковой ток (индуктивность): ~0.045 × 3 ≈ 0.135 А
SSR номинал: 2 А (резистивная нагрузка)
✅ Запас: 2 А / 0.135 А ≈ 15× — безопасно

Тепловыделение SSR

P_рассеяния ≈ 1.2 В × I_нагрузки
Пример: 1.2 В × 0.07 А ≈ 0.08 Вт — нагрев минимален, радиатор не требуется
⚠️ Но при длительной работе (>5 мин) обеспечить вентиляцию в корпусе

Выбор предохранителя

Рекомендация: 1 А, быстродействующий (тип F)
• Защищает от КЗ в катушке клапана
• Не срабатывает от пускового тока (0.135 А << 1 А)

🛡️ Меры безопасности
Аппаратные
Предохранитель 1 А на фазе перед SSR — обязательно
Заземление металлического корпуса клапана и электроблока
Герметичный бокс для высоковольтной части (отделение от низковольтной)
Термостойкие провода (ПВС, до 105 °С) для подключения клапана
Механический фильтр 100–200 мкм перед клапаном — защита от засорения
Программные
Тайм-аут аварийного закрытия клапана (макс. время работы)
Проверка состояния датчиков перед открытием клапана
Логирование всех переключений для отладки
Эксплуатационные
Тест на герметичность перед первым запуском
Периодический осмотр уплотнений (каждые 10 дистилляций)
Замена уплотнений при первых признаках износа

✅ Чеклист перед первым запуском

[ ] Измерить сопротивление катушки клапана (ожидается 1.5–3.5 кОм)
[ ] Проверить материал уплотнения (силикон → заменить на Viton)
[ ] Установить предохранитель 1 А на фазу перед SSR
[ ] Подключить RC-модуль параллельно клапану
[ ] Протестировать работу клапана на воздухе/воде без нагрева:
    - [ ] Клапан закрыт по умолчанию (без питания)
    - [ ] При подаче 220В — открывается, поток есть
    - [ ] При отключении 220В — закрывается, поток прекращается
[ ] Проверить герметичность в закрытом состоянии под давлением
[ ] Добавить в код тайм-аут аварийного закрытия
[ ] Обеспечить вентиляцию в корпусе электроники
[ ] Протестировать управление с ESP32 (GPIO → SSR → клапан)
[ ] Проверить логи: фиксация всех переключений клапана
