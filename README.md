## 🔌 Управление нагрузкой 220В (клапаны)

> Управление электромагнитными клапанами через твердотельное реле (SSR) с опторазвязкой.  
> 📄 Полная документация: [`SSR.md`](./SSR.md)

### 🧩 Компоненты

| Компонент | Назначение | Ключевые параметры |
|-----------|-----------|-------------------|
| **SSR-модуль** | Коммутация 220В с гальванической развязкой | 5В управление high trigger), 24–240В AC, 2А, Zero-Cross |
| **RC-снаббер + варистор** | Защита от выбросов при коммутации индуктивной нагрузки | 5–400В, до 1000Вт, параллельно клапану |
| **Клапан 1/4" 220В** | Перекрытие потока (головы/хвосты/отбор) | НЗ, нержавеющая сталь, уплотнение **Viton (FKM)** ✅ |
